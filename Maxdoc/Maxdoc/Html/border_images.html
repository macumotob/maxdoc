Border Images

With the CSS3 border-image property, you can set an image to be used as the border around an element.

  border: 10px solid transparent;
  border-image: 
   url(img/border.png) 30 round;
  width:300px;
  height:150px;
    Свойство border-image состоит из трех частей:
  • Image который будет использоваться для заполнения границы
  • Как, в каком месте делить image
  • Определния того, будет ли средняя секция повторяться или растягиваться

В примерах будет использоваться следующий image

Пример - середина картинки повторяется для создания границы: 

  border: 10px solid transparent;
  padding: 15px;
  /* Safari 3.1-5 */
  -webkit-border-image:
     url(border.png) 30 round;
 /* Opera 11-12.1 */ 
  -o-border-image:
     url(border.png) 30 round;
  border-image:
    url(border.png) 30 round;

Середина картинки растягивается для создания границы: 

  border: 10px solid transparent;
  padding: 15px;
  /* Safari 3.1-5 */
  -webkit-border-image:
     url(border.png) 30 stretch; 
  /* Opera 11-12.1 */
  -o-border-image: 
    url(border.png) 30 stretch;
  border-image: 
   url(border.png) 30 stretch;

Свойства границы

Property Description
border-imageA shorthand property for setting all the border-image-* properties
border-image-sourceSpecifies the path to the image to be used as a border
border-image-sliceSpecifies how to slice the border image
border-image-widthSpecifies the widths of the border image
border-image-outsetSpecifies the amount by which the border image area extends beyond the border box
border-image-repeatSpecifies whether the border image should be repeated, rounded or stretched

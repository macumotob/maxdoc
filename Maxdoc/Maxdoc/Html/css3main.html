Введение в CSS3

CSS3 является последним стандартом CSS.

CSS3 полностью совместим с более ранними версиями CSS.

Этот раздел ознакомит Вас с новыми возможностями CSS3.

CSS3 был разделен на модули. Он содержит "старую спецификацию CSS" (которая в свою очередь разбита на более мелкие модули). Кроме того , были добавлены новые модули. Вот наиболее важные из новых модулей CSS:
  • Selectors
  • Box Model
  • Backgrounds and Borders
  • Image Values and Replaced Content
  • Text Effects
  • 2D/3D Transformations
  • Animations
  • Multiple Column Layout
  • User

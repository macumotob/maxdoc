Закругленные углы / Rounded Corners

Используя свойство border-radius Вы можете закруглить углы любого элемента.

    Примеры:
  • 1. Закругленные углы элемента с определенным цветом фона (background-color). 
     #rcorners1 {
    border-radius: 25px;
    background: green;
    padding: 20px; 
    width: 300px;
    height: 150px; 
  }
  • 2. Закругления элемента с границей 
     #rcorners2 {
    border-radius: 25px;
    border: 2px solid green;
    padding: 20px; 
    width: 300px;
    height: 150px; 
}
  • 3. Закругления у элемента с image в качкстве фона 
     #rcorners3 {
   border-radius: 25px;
   border: 2px solid green;
   background: 
       url(img/round_corners.png);
   background-position: left top;
   background-repeat: repeat-x;
   padding: 20px; 
   width: 200px;
   height: 250px; 
}
    Свойство border-radius это сокращение для свойств
  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius
Если Вы укажите только одно значение для свойства border-radius, это значение будут применено для всех четырех свойств.
    Вы можете указать значение радиуса для каждого угла, вот правило:
  • Четыре значения: top-left, top-right, bottom-right, bottom-left.
  • Три значения: top-left, top-right и bottom-left, bottom-right.
  • Два значения: top-left и bottom-right , top-right и bottom-left.
  • Одно значение: все четыре угла имеют одинаковое закругление.

    Примеры:
  • 1.Черыте значения border-radius: 15px 50px 30px 5px; 
    #rcorners4 {
    border-radius:
     15px 50px 30px 5px;
    background: green;
    padding: 20px; 
    width: 300px;
    height: 150px; 
}
  • 2. Три значения border-radius: 15px 250px 30px 
    #rcorners5 {
  border-radius: 15px 250px 30px;
  background: green;
  padding: 20px;
  width: 300px;
  height: 150px;
}
  • 3. Два значения: border-radius: 15px 50px 
     #rcorners6 {
    border-radius: 15px 50px;
    background: green;
    padding: 20px; 
    width: 300px;
    height: 150px; 
}

Вы можете создавать элиптические закругления

    Примеры:
  • border-radius: 150px/75px; 
      #rcorner7 {
    border-radius: 
         150px/75px;
    background: green;
    padding: 20px; 
    width: 300px;
    height: 150px; 
        }
  • border-radius: 75px/150px; 
    #rcorners8 {
    border-radius: 75px/150px;
    background: green;
    padding: 20px; 
    width: 300px;
    height: 150px; 
}
  • border-radius: 50%; 
        #rcorners9 {
        border-radius: 50%;
        background: green;
        padding: 20px; 
        width: 300px;
        height: 150px;
    }

Синтаксис CSS3

border-radius: 1-4 length|% / 1-4 length|%|initial|inherit;

  • 1. border-radius: initial; 
          #rcorners10 {
        border-radius: initial;
        background: green;
        padding: 20px; 
        width: 300px;
        height: 150px;
    }
  • 2. border-radius: inherit; 
          #rcorners10 {
        border-radius: inherit;
        background: green;
        padding: 20px; 
        width: 300px;
        height: 150px;
    }
  • 3. border-radius: 25% 30%/ 12% 45%; 
      #rcorners10 {
    border-radius: 25% 30%/ 12% 45%;
    background: green;
    padding: 20px; 
    width: 300px;
    height: 150px;
}
  • 4. border-radius: 2em 3em / 5em 0.4em; 
      #rcorners10 {
    border-radius:
       2em 3em / 5em 0.4em;
    background: green;
    padding: 20px; 
    width: 300px;
    height: 150px;
}
  • 5. border-bottom-right-radius: 4em 5em 
      #rcorners10 {
    border-bottom-right-radius: 
        4em 5em;
    background: green;

    width: 300px;
    height: 150px;
